# muon-dark
muon-dark vim theme

Theme is a copy of the https://github.com/gregsexton/Muon theme with modified background color to dark.

# Installation
Assuming you are using a [vim-plug](https://github.com/junegunn/vim-plug)

`Plug 'gs/muon-dark'`

`PlugInstall`

`colorscheme mount-dark`

and enjoy.
